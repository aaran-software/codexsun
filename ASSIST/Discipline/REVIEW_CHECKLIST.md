# Review Checklist

## Priorities

1. correctness
2. clarity
3. maintainability
4. repo-accurate documentation

## Checklist

1. Does the change match `ASSIST/Documentation/ARCHITECTURE.md`?
2. Is the code simpler or at least no more confusing than before?
3. Are component and styling responsibilities still clear?
4. Were relevant validation steps run?
5. If validation was skipped, is the gap stated explicitly?
