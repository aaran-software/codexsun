# Prompt 050 — Audit Logs & System Monitoring

Implement a **complete Audit Logs and System Monitoring module** for the Codexsun platform.

The goal is to **track all system activities, user actions, security events, and application errors** across the platform.

This module must work **centrally across all modules** including:

* Authentication
* Vendors
* Orders
* Inventory
* Payments
* Settings
* Admin actions

The system must support **audit trails, system logs, error logs, and login history tracking**.

# 2 Database Tables

Create tables **only if they do not exist**.

---

## audit_logs

Tracks **user actions across the platform**.

```
audit_logs
-----------
Id (Guid, PK)
UserId (Guid, nullable)
Action (nvarchar 200)
EntityType (nvarchar 100)
EntityId (nvarchar 100)
Module (nvarchar 100)
OldValues (nvarchar(max))
NewValues (nvarchar(max))
IPAddress (nvarchar 100)
UserAgent (nvarchar 500)
CreatedAt (datetime)
```

Examples:

```
User updated product
Admin deleted vendor
Vendor changed order status
Admin updated settings
```

---

## system_logs

Tracks **system events and internal operations**.

```
system_logs
------------
Id (Guid, PK)

Message (nvarchar(max))

```

Examples:


---

## AuditLogService Responsibilities

* log user actions
* capture entity changes
* store before/after values
* attach user and request metadata

Example usage

```
AuditLogService.Log(
  userId,
  "Update Product",
  "Product",
  productId,
  "Catalog",
  oldValues,
  newValues
)
```

---

## ErrorLogService Responsibilities

Capture exceptions globally.

Integrate with **ASP.NET middleware**.

Example:

```
app.UseMiddleware<ErrorLoggingMiddleware>()
```

Middleware responsibilities

```
capture exception
log stacktrace
store request path
store user id
```

---

## SystemLogService Responsibilities

Used for internal operations.

Example:

```
SystemLogService.Log(
   "InventoryService",
   "RecalculateStock",
   "Stock recalculation started",
   "Info"
)
```

---

## LoginHistoryService Responsibilities

Track

```
login success
login failure
logout
account lock
```

Used inside **Auth module**.
