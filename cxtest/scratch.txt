# Prompt — Storefront Frontend (Customer Ecommerce UI)

Build the **complete customer storefront frontend** for the Codexsun multi-vendor ecommerce platform.

Stack:

```
React
TypeScript
React Router
Axios
React Query
TailwindCSS
Redux Toolkit (or Zustand)
```

The storefront must connect to the **existing ASP.NET Core backend APIs**.

The storefront should support:

```
Home
Product browsing
Search
Category pages
Product page
Cart
Checkout
Orders
User account
Wishlist
Reviews
Vendor stores
```

---
