# RETURN WORKFLOW

Standard flow:

```
Customer requests return
↓
Admin reviews request
↓
Return approved
↓
Customer ships product
↓
Warehouse receives item
↓
Inspection performed
↓
Item restocked
↓
Refund issued
```
