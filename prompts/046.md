PROMPT 046

Implement a centralized Company and Application Settings module used to manage software branding, billing information, and global configuration values.

The goal is to ensure that company information such as logo, display name, billing name, address, and other configuration values are NOT hardcoded anywhere in the system.

This module will act as the single source of truth for platform-level configuration.
