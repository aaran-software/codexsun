UserId
Username
Role
Permissions

JWT signature validation

Invalid token rejected
Tampered token rejected

LOGIN ATTACK TESTS

' OR 1=1 --

Expected result:

Authentication must fail.

---

XSS Injection attempt:

Example:

<script>alert(1)</script>

Expected result:

Rejected or sanitized.

Brute force attack simulation.

Send repeated login attempts.

Expected result:

Rate limiter triggers.

HTTP status:

REFRESH TOKEN SECURITY TESTS

Verify refresh token behavior.

Test cases:

Old refresh token revoked after refresh
Refresh token cannot be reused
Expired refresh token rejected
Invalid refresh token rejected

SESSION SECURITY

Verify token isolation.

Test cases:

Access token cannot impersonate another user

Revoked token rejected

AUTHORIZATION TESTS

Verify protected endpoints.
