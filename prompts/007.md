OBJECTIVE

Create the **Common Master Data module** used across the platform.

These masters provide reusable datasets used by:

• product forms
• order forms
• address forms
• vendor forms
• logistics forms

All master data must support:

• Autocomplete UI
• Reusability
• Active / inactive toggle

These modules must be scalable and reusable across domains.
